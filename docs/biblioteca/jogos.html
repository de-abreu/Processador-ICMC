Jogos

Aqui encontram-se todos os jogos conhecidos produzidos para o processador ICMC, alguns deles recomendados.

Como jogar​

Para executar os seguintes jogos, obtenha um simulador e o provenha a este os respectivos arquivos .mif, disponibilizados abaixo. Ou mesmo os instale em uma placa FPGA.

Jogos recomendados​

Frogger Demake​

Uma implementação do jogo Frogger (KONAMI, 1981) para o processador do ICMC.

Arquivos​

frogger.mif charmap.mif

  • Como jogar
  • Ficha Técnica

Controles​

  • W: move para frente
  • A: move para a esquerda
  • S: move para trás
  • D: move para a direita

Pontuação​

  • 1 pt: para cada salto adiante
  • 5 pt: para cada sapo que consegue atravessar
  • 100 pt: ao atravessar todos os sapos
  • 1 pt: para cada segundo restante

Autores: Augusto Pereira, Guilherme de Abreu, Felipe Machado

Ano de produção: 2024

# Montador

Software responsável pela tradução das [instruções em linguagem assembly](/docs/linguagem-assembly/instrucoes) a sua forma correspondente em código binário para execução pela placa FPGA.

Por exemplo:

<Tabs>
    <TabItem value="Assembly">
        ```asmatmel
        ; Prints "Hello Mom" at the top left corner of the screen and halts.

        jmp main

        hello: string "Hello Mom"

        main:
            loadn r0, #0         ; r0 = 0
            loadn r1, #hello     ; r1 = Pointer to the string to be printed
            loadn r2, #0         ; r2 = Screen position

            printLoop:
                loadi r3, r1     ; Retrieve character to be printed
                cmp r0, r3       ; and check if it is the terminator character.
                jeq endLoop      ; Escape if it is.
                outchar r3, r2   ; Otherwise, print it and continue.
                inc r1           ; Increment string pointer
                inc r2           ; Increment screen position
                jmp printLoop
            endLoop:
                halt
            ```
        </TabItem>

## Instalação
O montador encontra-se disponível em duas versões, cujo código fonte pode ser baixado em um arquivo zip por meio dos seguintes links:

- [`v1.0.0 stable`](https://github.com/de-abreu/Processador-ICMC/raw/master/assembler/assembler-stable.zip) (recomendado)
- [`v2.1.0 beta`](https://github.com/de-abreu/Processador-ICMC/raw/master/assembler/assembler-stable.zip)

Para instalar execute os seguintes comandos:

<Tabs>
    <TabItem value="stable">
        ```bash
        unzip assembler-stable.zip # Extrai a pasta "stable"
        cd stable # Acessa a pasta stable
        gcc *.c -o montador # Compila o código-fonte para gerar o executável "montador"
        ```
    </TabItem>
    <TabItem value="beta">
        ```bash
        unzip assembler-beta.zip # Extrai a pasta "beta"
        cd beta# Acessa a pasta beta
        gcc *.c -o montador # Compila o código-fonte para gerar o executável "montador"
        ```
    </TabItem>
</Tabs>

Com isso você terá em mãos o executável `montador`.Leve-o ao diretório em que estiver trabalhando, conforme sua conveniência.

## Uso

Considerando um arquivo assembly `jogo.asm`, execute o seguinte comando opara obter um arquivo binário `jogo.mif` (a operação é análoga para quaisquer outros nomes de arquivo):


<Tabs>
    <TabItem value="1.0.0" label="stable">
        ```bash
        ./montador jogo.asm jogo.mif
        ```
    </TabItem>
    <TabItem value="2.1.0" label="beta">
        ```bash
        ./montador jogo.asm
        ```
    </TabItem>
</Tabs>

Se não houverem erros de sintaxe, o arquivo `jogo.mif` será gerado. Doutra forma, uma mensagem de erro correspondente será emitida.

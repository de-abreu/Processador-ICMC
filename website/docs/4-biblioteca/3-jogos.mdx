# Jogos

Aqui encontram-se todos os jogos conhecidos produzidos para o processador ICMC, alguns deles recomendados.
## Como jogar

Para executar os seguintes jogos, obtenha um [simulador](/docs/componentes/simulador) e o provenha a este os respectivos arquivos `.mif`, disponibilizados abaixo. Ou mesmo os instale em uma [placa FPGA](/docs/componentes/processador).

<DynamicSection>

## Jogos recomendados
<details><summary>

### Frogger Demake
Uma implementação do jogo [Frogger (KONAMI, 1981)](https://invidious.privacyredirect.com/watch?v=WNrz9_Fe-Us) para o processador do ICMC.

</summary>

![Captura de tela do jogo Frogger Demake](/img/games/frogger-demake.png)
#### Arquivos

**<Highlight color="#26762b">[frogger.mif](https://raw.githubusercontent.com/de-abreu/frogger-demake/main/frogger.mif)</Highlight>**
**<Highlight color="#23238e">[charmap.mif](https://github.com/de-abreu/frogger-demake/raw/main/graphics/charmap.mif)</Highlight>**

<Tabs>
<TabItem value="Como jogar">

#### Controles

- **W:** move para frente
- **A:** move para a esquerda
- **S:** move para trás
- **D:** move para a direita

#### Pontuação
- **1 pt:** para cada salto adiante
- **5 pt:** para cada sapo que consegue atravessar
- **100 pt:** ao atravessar todos os sapos
- **1 pt:** para cada segundo restante

</TabItem>
<TabItem value="Ficha Técnica">

**Autores:** [Augusto Pereira](https://github.com/gutokb), [Guilherme de Abreu](https://github.com/de-abreu), [Felipe Machado](https://github.com/FelipeCarneiroMachado)

**Ano de produção:** 2024

**<LinkItem item = {{ label: "Site do Projeto", href: "https://github.com/de-abreu/frogger-demake/tree/main"}} />**

</TabItem>
</Tabs>
</details>

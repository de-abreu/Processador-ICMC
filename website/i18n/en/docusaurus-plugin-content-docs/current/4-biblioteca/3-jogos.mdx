# Games

Here are all the known games produced for the ICMC processor, some of them recommended.
## How to Play

To run the following games, obtain a [simulator](/docs/componentes/simulador) and provide it with the respective `.mif` files, made available below. Or install them on an [FPGA board](/docs/componentes/processador).

<DynamicSection>

## Recommended Games
<details><summary>

### Frogger Demake
An implementation of the game [Frogger (KONAMI, 1981)](https://invidious.privacyredirect.com/watch?v=WNrz9_Fe-Us) for the ICMC processor.

</summary>

![Screenshot of the game Frogger Demake](/img/games/frogger-demake.png)
#### Files

**<Highlight color="#26762b">[frogger.mif](https://raw.githubusercontent.com/de-abreu/frogger-demake/main/frogger.mif)</Highlight>**
**<Highlight color="#23238e">[charmap.mif](https://github.com/de-abreu/frogger-demake/raw/main/graphics/charmap.mif)</Highlight>**

<Tabs>
<TabItem value="How to Play">

#### Controls

- **W:** move forward
- **A:** move left
- **S:** move back
- **D:** move right

#### Scoring
- **1 pt:** for each jump forward
- **5 pt:** for each frog that reaches the other side
- **100 pt:** for completing all levels
- **1 pt:** for each remaining second

</TabItem>
<TabItem value="Technical Details">

**Authors:** [Augusto Pereira](https://github.com/gutokb), [Guilherme de Abreu](https://github.com/de-abreu), [Felipe Machado](https://github.com/FelipeCarneiroMachado)

**Year of production:** 2024

**<LinkItem item = {{ label: "Project Website", href: "https://github.com/de-abreu/frogger-demake/tree/main"}} />**

</TabItem>
</Tabs>
</details>
</DynamicSection>
